PROJECTTRANSFER UTF-8 QUICK FIX

1. Copy run_built_exe.bat to the root ProjectTransfer source folder.
2. Replace the old file.
3. Start the application only through run_built_exe.bat.

No EXE rebuild is required for this quick fix.
The launcher forces UTF-8 for the GUI and ProjectTransferWorker.exe.

For a permanent direct-EXE fix, use the full v2.7 package and rebuild.
